# The Shahram File

My portfolio, filed as evidence. Live at **[the-shahram-file.vercel.app](https://the-shahram-file.vercel.app)**.

Scroll through the sealed door of Evidence Room B-2 and you land in front of a corkboard holding
everything the department has on me: field reports (experience), exhibits (projects), records
(education and skills), all connected with red string.

## Things to try

- **Drag the evidence.** Every card is draggable; the red string follows in real time.
- **Open the folders.** Click any card for its full confidential dossier.
- **Find the UV lamp.** Some notes are written in invisible ink.
- **File a tip.** The tip line at the bottom of the board delivers straight to my inbox.

## Stack

One `index.html`. No frameworks, no build step. Scroll-scrubbed 3D CSS door, SVG string physics,
Web3Forms contact, pure vanilla everything.
